# 3D Model Assets

This folder contains locally served `.glb` models for the virtual learning scenes.

Current starter assets:

- `animals/parrot.glb` - bird model used for the toucan/bird habitat marker.
- `animals/fox.glb` - quadruped animal model used as a temporary predator stand-in for the tiger marker.

To upgrade realism further, replace or add exact models here and update
`client/src/scene/AnimalHabitatScene.ts` in `animalModelPlacements`.

Recommended future filenames:

- `animals/tiger.glb`
- `animals/monkey.glb`
- `plants/rainforest-tree.glb`
